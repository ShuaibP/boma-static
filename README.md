# Boma Landing Page

Single-page informational website for a fintech concept where users earn cashback from partner retailers and invest it through different risk-based fund options.

## Tech stack

- HTML5
- CSS3
- Vanilla JavaScript
- CDN-hosted open source libraries:
  - Bootstrap 5
  - Bootstrap Icons
  - AOS (Animate On Scroll)

## Project files

- `index.html` - Main page structure and content
- `styles.css` - Custom styles and responsive behavior
- `script.js` - Interactions (AOS initialization, navbar behavior, waitlist form feedback)

## Run locally

1. Clone or download this repository.
2. Open `index.html` directly in your browser.

No build step is required.

## Deploy to GitHub Pages

1. Create a new GitHub repository (example: `your-username/boma-landing-page`).
2. Add these files to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Push to the `main` branch.
4. In GitHub, open **Settings -> Pages**.
5. Under **Build and deployment**:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
6. Save changes and wait for deployment.
7. Your site will be available at:
   - `https://your-username.github.io/your-repository-name/`

## Notes

- Replace retailer placeholders in the Partner Retailers section with official logos.
- Footer links are placeholders and should be updated with real pages when available.
- Waitlist form currently provides client-side feedback only and does not submit data to a backend.
